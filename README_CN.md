# Unity UI Toolbar

Unity Editor 的强大 UI 工具栏扩展，为 Scene 视图提供快速对齐、分组和组件创建功能。

## ✨ 功能特性

### 🎯 UI 对齐工具栏 (UI Alignment Toolbar)

在 Scene 视图中快速对齐和排列 UI 元素，无需手动调整坐标。

#### 分组功能
- **组合** (Group) - 为选中对象创建父级空对象
- **取消组合** (Ungroup) - 解散组合，保留子对象
- **均匀行间距** - 在选中对象之间创建均匀的垂直间距
- **均匀列间距** - 在选中对象之间创建均匀的水平间距
- **网格布局** - 将选中对象按网格排列（支持自定义行列数和间距）

#### 对齐功能
- **左对齐** - 将选中的 UI 元素左边缘对齐
- **水平居中** - 将选中的 UI 元素水平居中对齐
- **右对齐** - 将选中的 UI 元素右边缘对齐
- **顶部对齐** - 将选中的 UI 元素顶部对齐
- **垂直居中** - 将选中的 UI 元素垂直居中对齐
- **底部对齐** - 将选中的 UI 元素底部对齐

### 🎨 UI 组件工具栏 (UI Component Toolbar)

快速切换常用 UI 组件的创建。

支持的组件：
- **Image** - 图片组件
- **Text** - 文本组件
- **Input Field** - 输入框组件
- **Button** - 按钮组件
- **Toggle** - 开关组件
- **Slider** - 滑动条组件
- **Scrollbar** - 滚动条组件

## 📦 安装

### 方式 1：Unity Package Manager (推荐)

需要支持 git 包路径查询参数的 Unity 版本（Unity >= 2019.3.4f1, Unity >= 2020.1a21）。  
你可以将 `https://github.com/Naipa-Games/unity-ui-toolbar.git` 添加到包管理器。

### 方式 2：手动安装

- 从发布页面下载最新的 .unitypackage 文件。
- 双击文件导入到你的项目中。

### 启用工具栏

1. 打开 Scene 视图
2. 点击 Scene 视图右上角的 `⋮` (更多选项)
3. 在 `Overlay Menu` 菜单中找到并启用：
   - **UI Alignment Toolbar** - 对齐和分组工具
   - **UI Component Toolbar** - 组件创建工具

### 使用分组功能

**创建组：**
1. 选中多个对象
2. 点击 Group 按钮
3. 系统会创建一个父级对象包含所有选中对象

**取消组：**
1. 选中组对象
2. 点击 Ungroup 按钮
3. 子对象将移至组的父级，组对象将被删除

### 对齐 UI 元素

**单个对象：**
- 选中一个 RectTransform 对象
- 点击对齐按钮，对象将相对于其父级对齐

**多个对象：**
- 选中多个 RectTransform 对象
- 点击对齐按钮，所有对象将相对于第一个选中的对象对齐

### 使用间距功能

**均匀行/列间距：**
1. 选中多个对象（至少 2 个）
2. 点击间距按钮打开设置窗口
3. 设置间距值（像素）
4. 点击 Apply 应用

### 使用网格布局

1. 选中要排列的对象
2. 点击 Grid Layout 按钮
3. 在弹出窗口中设置：
   - 列数
   - 行数
   - 水平间距
   - 垂直间距
4. 点击 Apply 应用布局

### 创建 UI 组件

1. 在 Hierarchy 中选中 Canvas 或 UI 父对象，如果不选中则以 Canvas 为父对象。
2. 点击 Component Toolbar 中点击对应的组件按钮后，再次点击场景区域进行创建。
